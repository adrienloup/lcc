# Life Cycle Calculator

What is the impact of your website on the planet?

This calculator is an eco-design and Life Cycle Assessment (LCA) tool to help you eco-design your zero carbon website. The calculation gives a result per user, per year and per raw data. The tool uses several key data: website traffic, energy intensity, carbon intensity of electricity and energy source used.

[Demo](https://adrienloup.github.io/lcc/)

## Installing

Node
```
nvm use
```
npm
```
npm install
```

## Development server

Starts a local web server with [HMR](https://vitejs.dev/guide/features.html#hot-module-replacement) for development

```
npm run dev
```

## Production build

Builds the project, and outputs to the folder `./dist`

```
npm run build
```

## Testing the App Locally

Start a local web server that serves the built solution from `./dist` for [previewing](https://vitejs.dev/guide/static-deploy.html#testing-the-app-locally)

```
npm run preview
```
